# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-14

- Initial release.
- Declarative DSL macros for GPUI.
- Style token support for layout, spacing, typography, borders, rounding, shadows, and overflow.
- Event sugar for common GPUI handlers.
- `list(...)` DSL for `uniform_list`.
